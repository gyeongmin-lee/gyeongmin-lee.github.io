How It Was Built

I developed this application as a way to practice creating a streamlined onboarding process using React. The development process took about 10 hours in total, including 1 hour of research on other onboarding processes, 2 hours of creating a mockup in Figma, and 7 hours of building the application itself.

The application is a basic React App created with Next.js. I utilized Redux Toolkit, a library that simplifies the setup of a Redux store and provides a set of conventions for structuring the store and its associated logic, to manage the application’s state.

Features

  • Choose a weekly meal plan
  • Add meals to the cart either automatically or manually
  • Filter meals by meal property
  • Sign up form with simple auto-complete

Built With

  • React
